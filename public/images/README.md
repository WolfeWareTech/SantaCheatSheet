# Santa's Cheat Sheet Images

This directory should contain the following images for the application:

## Required Images

### 1. Santa.png
- **Location**: `/public/images/Santa.png`
- **Description**: Santa Claus character illustration holding a scroll/list
- **Recommended size**: 140px width × 160px height (or proportional)
- **Usage**: Displayed in the top-right corner of the header
- **Style**: Cartoon/illustrated Santa with a friendly appearance

### 2. SantaStamp.png
- **Location**: `/public/images/SantaStamp.png`
- **Description**: Circular stamp/seal with "GENUINE MADE BY SANTA CLAUS NORTH POLE ILLINOIS" text and Santa's sleigh silhouette
- **Recommended size**: 120px × 120px
- **Usage**: Displayed in the bottom-right corner of the signature area
- **Style**: Circular stamp design with vintage/official seal appearance

## Adding Your Images

1. Place your `Santa.png` and `SantaStamp.png` files in this directory
2. Ensure the images are PNG format with transparent backgrounds for best results
3. The application will automatically load them from this location
4. If images are not present, the layout will still work but the image areas will be empty

## Image Sources

You can create or source these images from:
- Custom illustrations
- Licensed stock images
- Public domain Christmas clipart
- AI-generated images (with appropriate licensing)

Make sure you have the right to use any images you add to this project.
